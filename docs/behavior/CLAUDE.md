# Behavior docs — writing rules

Scoped rules for the product behaviour specifications in this directory
(`docs/behavior/`). These apply on top of the repository-root `CLAUDE.md`.

## Terminology

- **Do not use the word "mail" to describe messages.** It is misleading and
  unconventional for this product. Prefer plain, conventional terms —
  "message", "direct message", or the concrete mechanism (e.g. "sealed,
  single-addressed message", "gift-wrapped event"). This applies to
  reactions, direct messages, and any other message-carrying channel described
  in these documents.
